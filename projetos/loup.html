Loup

quando: 2019.2
o que: UX design
onde: ESPM
como: Adobe XD

briefing:

Desenvolver um aplicativo que tenha algum impacto sustentável. Nesse caso, desenvolver um aplicativo que funcione como uma rede para conectar pessoas que desejem realizar alguma doação com quem necessita.

desenvolvimento:

Projeto iniciou-se com um brainstorming para definir que problema o aplicativo solucionaria a fim de gerar algum impacto positivo para a sociedade. Foi definido que seria desenvolvida uma rede para conectar pessoas com itens para doação com pessoas que desejam possuir-los. Assim, iniciou-se o estudos das ferramentas necessárias para o funcionamento do aplicativo - como a página de doações, divisão de categorias, chat, entre outros. Depois, foi desenvolvido os Wireframes e a arquitetura de informação, para que, por último, a interface fosse criada.

sobre o aplicativo:

Loup é um protótipo de aplicativo que tem como propósito conectar pessoas que desejem realizar doações de bens, com pessoas que os procurem, tendo em vista que muitas vezes é difícil achar lugares que aceitem doações dos mais variados itens. O objetivo é incentivar a reutilização de objetos que já não são mais úteis para o atual dono, evitando o descarte de objetos em bom estado, que podem ser úteis para outras pessoas. O lobo, é símbolo da compaixão, sentimento essencial para a atividade de doação. Por esse motivo foi escolhido para representar a marca. O nome foi estrategicamente traduzida para o francês, Loup, pois tem a pronúncia muito similar à "Loop", palavra utilizada para referir-se a movimentos circulares, rotatividade. Assim como o que o aplicativo se baseia em, a rotatividade de bens entre pessoas.
